# AX Starter Kit

> 기업 AI 전환(AX)을 4주 만에 시작하는 실전 툴킷

AI 도구를 도입하는 것만으로는 아무것도 바뀌지 않습니다.
**AX Starter Kit**은 실제 기업 도입 경험을 바탕으로, 비개발자부터 개발자까지 AI를 "일하는 방식"에 정착시키는 실행 가이드입니다.

## 이런 분들을 위해 만들었습니다

- AI 전환을 추진해야 하는 **팀 리더 / 경영진**
- 사내 AI 도입을 담당하는 **AX 담당자 / FDE**
- "AI 써보라는데 뭐부터 해야 하지?" 라는 **현업 담당자**

## 구성

```
ax-starter-kit/
├── playbook.md              ← 4주 실행 플레이북 (여기서 시작하세요)
├── 01-diagnosis/            ← 현황 진단 도구
├── 02-onboarding/           ← 온보딩 가이드 + 프롬프트 템플릿
│   ├── non-dev/             ← 비개발자용
│   └── dev/                 ← 개발자용 (예정)
├── 03-security/             ← 보안 거버넌스
├── 04-measurement/          ← 성과 측정 (예정)
└── case-studies/            ← 실제 도입 사례
```

## 빠른 시작

1. [`playbook.md`](playbook.md)를 읽고 4주 전체 흐름을 파악하세요
2. Week 0의 "왜 AX를 해야 하는가"로 팀/경영진 공감대를 만드세요
3. [`01-diagnosis/`](01-diagnosis/)로 우리 팀 업무를 진단하세요
4. Week 1부터 순서대로 실행하세요

## 다른 AI 도입 가이드와 다른 점

| | 일반 가이드 | AX Starter Kit |
|---|---|---|
| 기반 | 이론/컨설팅 | **실제 기업 도입 경험** |
| 대상 | 개발자 중심 | **비개발자부터 시작** |
| 형태 | 보고서/슬라이드 | **바로 쓸 수 있는 도구** |
| 보안 | 언급만 | **정책 템플릿 제공** |
| 성과 | "효과 있다" | **측정 프레임워크 제공** |
