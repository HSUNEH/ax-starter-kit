# 데이터 분석/정리 프롬프트 템플릿

> 복사 → [괄호] 수정 → AI에 붙여넣기

---

## 기본형: 데이터 정리 및 요약

```
아래 데이터를 분석하고 요약해주세요.

[데이터]
(표, 숫자, 텍스트 등 원본 데이터를 붙여넣으세요)

[분석 요청]
- 주요 트렌드나 패턴
- 이상치나 주목할 점
- [추가 분석 요청]

[출력 형식]
- 핵심 인사이트 3~5개 (불릿포인트)
- 데이터 요약 표
- 시사점 및 제안
```

---

## 활용 예시

### Before (기존 방식)

**상황**: 엑셀에서 300행 데이터를 정리하고 트렌드를 파악하는 데 2시간 소요.

### After (AI 적용)

**프롬프트:**
```
아래는 최근 6개월간 고객 문의 데이터입니다.
분석하고 인사이트를 도출해주세요.

[데이터]
월, 총 문의, 제품 문의, 배송 문의, 반품 문의, 기타
10월, 1250, 380, 420, 310, 140
11월, 1180, 350, 390, 290, 150
12월, 1890, 520, 680, 450, 240
1월, 1420, 410, 450, 380, 180
2월, 1350, 390, 430, 350, 180
3월, 1280, 370, 400, 330, 180

[분석 요청]
- 월별 추이 및 계절성
- 문의 유형별 비율 변화
- 12월 급증 원인 추정
- 개선이 필요한 영역

[출력 형식]
- 핵심 인사이트 (불릿포인트)
- 월별 트렌드 요약 표
- 권장 조치사항
```

**결과**: 15분 만에 인사이트 + 권장 조치 도출. **총 20분** (기존 2시간 → 83% 절감)

> ⚠️ **주의**: AI가 제시하는 수치와 분석은 반드시 원본 데이터와 대조하세요. 특히 합계, 비율 등 계산이 들어간 부분은 검증이 필요합니다.

## 변형: 엑셀 수식/작업 요청

```
엑셀에서 아래 작업을 수행하는 방법을 알려주세요.

[현재 상태]
- 데이터 위치: [시트명, 셀 범위]
- 열 구성: [A열: 날짜, B열: 매출, C열: 지역, ...]

[원하는 결과]
[구체적으로 어떤 결과물이 필요한지]

[출력 형식]
- 단계별 설명
- 사용할 수식 (복사 가능한 형태)
- 피벗테이블이 필요한 경우 설정 방법
```

---

## 변형: 텍스트 데이터 분류/정리

```
아래 텍스트 데이터를 분류하고 정리해주세요.

[데이터]
(설문 응답, 고객 피드백, 자유 형식 텍스트 등을 붙여넣으세요)

[분류 기준]
- [카테고리1]: [설명]
- [카테고리2]: [설명]
- [카테고리3]: [설명]
(기준이 없으면 "적절한 카테고리를 만들어서 분류해주세요"라고 해도 됩니다)

[출력 형식]
- 카테고리별 건수와 비율
- 카테고리별 대표 의견 2~3개
- 전체 요약 및 시사점
```

---

## 변형: 비교 분석

```
아래 두 데이터를 비교 분석해주세요.

[데이터 A: 설명]
(데이터 붙여넣기)

[데이터 B: 설명]
(데이터 붙여넣기)

[비교 기준]
- [기준1]
- [기준2]
- [기준3]

[출력 형식]
- 비교 요약 표
- 주요 차이점 (불릿포인트)
- 결론 및 제안
```

> 💡 **팁**: 데이터가 엑셀에 있다면, 관련 셀을 선택 → 복사 → AI에 붙여넣기 하면 표 형태로 인식합니다. 스크린샷보다 텍스트 복사가 훨씬 정확합니다.
