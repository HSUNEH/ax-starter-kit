# 이메일/커뮤니케이션 프롬프트 템플릿

> 복사 → [괄호] 수정 → AI에 붙여넣기

---

## 기본형: 업무 이메일

```
아래 상황에 맞는 업무 이메일을 작성해주세요.

[상황]
- 받는 사람: [직책/관계 — 예: 거래처 담당자, 팀장님, 타팀 동료]
- 목적: [요청/공유/회신/사과/안내]
- 핵심 내용: [전달할 메시지 요약]
- 톤: [정중하고 격식 있게 / 친근하지만 프로페셔널하게]
- 길이: [3~5문장 / 짧게]
```

---

## 활용 예시

### Before (기존 방식)

**상황**: 거래처에 일정 변경을 요청하는 메일. 어떻게 정중하게 쓸지 고민하며 20분 소요.

### After (AI 적용)

**프롬프트:**
```
아래 상황에 맞는 업무 이메일을 작성해주세요.

[상황]
- 받는 사람: 거래처 A사 김 과장님
- 목적: 다음 주 화요일 미팅 일정을 목요일로 변경 요청
- 핵심 내용: 내부 일정 충돌로 불가피하게 변경 요청. 양해 부탁.
- 톤: 정중하고 격식 있게
- 길이: 5~7문장

추가 조건:
- "죄송합니다"를 남발하지 말고 1번만 사용
- 대안 일정(목요일 같은 시간)을 제안
```

**결과**: 3분 만에 초안 완성. **총 5분** (기존 20분 → 75% 절감)

## 변형: 슬랙/팀즈 메시지

```
아래 내용을 슬랙 메시지로 작성해주세요.

[상황]
- 채널: [팀 채널 / DM / 전사 공지]
- 내용: [전달할 내용]
- 톤: [캐주얼 / 공식]

[원칙]
- 한 화면에 보이는 길이 (스크롤 없이)
- 핵심을 첫 줄에
- 필요하면 불릿포인트 사용
```

---

## 변형: 영문 이메일

```
아래 한국어 내용을 영문 비즈니스 이메일로 작성해주세요.

[상황]
- 받는 사람: [해외 파트너/외국인 동료]
- 관계: [처음/기존 거래처/동료]

[한국어 내용]
(전달할 내용을 한국어로 적으세요)

[원칙]
- 비즈니스 영어 톤
- 간결하게 (3~5문장)
- 문화적으로 자연스럽게 (직역 금지)
```

---

## 변형: 까다로운 상황 대응

```
아래 상황에서 상대방의 기분을 상하지 않으면서 [거절/요청/피드백]하는 이메일을 작성해주세요.

[상황]
(구체적인 상황을 설명하세요)

[원칙]
- 상대방의 입장을 먼저 인정
- 이유를 간결하게 설명
- 대안을 제시
- 관계 유지가 최우선
```

> 💡 **팁**: 까다로운 커뮤니케이션일수록 AI 초안이 유용합니다. 감정이 섞이지 않은 객관적 초안에서 시작하면 훨씬 수월합니다.
