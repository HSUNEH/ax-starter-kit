# 회의록 요약 프롬프트 템플릿

> 복사 → [괄호] 수정 → AI에 붙여넣기

---

## 기본형: 회의록 작성

```
아래 회의 내용을 정리된 회의록으로 작성해주세요.

[회의 정보]
- 회의명: [회의 제목]
- 일시: [날짜, 시간]
- 참석자: [이름/직책]

[회의 내용]
(회의 중 메모한 내용, 녹음 전사 텍스트, 또는 기억나는 내용을 그대로 붙여넣으세요.
정리되지 않은 상태여도 괜찮습니다.)

[출력 형식]
1. 핵심 논의 사항 (3줄 이내 요약)
2. 안건별 논의 내용
3. 결정 사항
4. 액션 아이템 (담당자, 기한 포함)
5. 다음 회의 일정/안건
```

---

## 활용 예시

### Before (기존 방식)

**상황**: 1시간 회의 후 회의록 정리에 30분 소요. 뭘 빠뜨렸는지 불안함.

### After (AI 적용)

**프롬프트:**
```
아래 회의 내용을 정리된 회의록으로 작성해주세요.

[회의 정보]
- 회의명: Q2 마케팅 전략 회의
- 일시: 2026-03-28 14:00~15:00
- 참석자: 김팀장, 이대리, 박사원, 최매니저

[회의 내용]
김팀장이 Q1 성과 리뷰함. 전체적으로 목표 대비 90% 달성.
SNS 캠페인이 잘됐고 검색광고는 부진.
이대리가 Q2에는 영상 콘텐츠 늘리자고 제안.
예산은 전분기와 동일. 박사원이 인플루언서 협업 제안했는데
김팀장이 예산 내에서 가능한지 다음주까지 알아보라고 함.
최매니저가 CRM 데이터 분석 결과 기존 고객 재구매율이 낮다고 보고.
리텐션 캠페인 별도로 기획하기로 함. 최매니저 담당.
다음 회의는 4월 4일.

[출력 형식]
1. 핵심 논의 사항 (3줄 이내 요약)
2. 안건별 논의 내용
3. 결정 사항
4. 액션 아이템 (담당자, 기한 포함)
5. 다음 회의 일정
```

**결과**: 5분 만에 깔끔한 회의록 완성. **총 7분** (기존 30분 → 77% 절감)

## 변형: 녹음 전사본 → 회의록

```
아래는 회의 녹음을 텍스트로 변환한 전사본입니다.
이 내용을 아래 형식의 회의록으로 정리해주세요.

[원칙]
- 중복/잡담/불필요한 내용은 제거
- 발언자별 핵심 의견만 추출
- 합의된 사항과 미결 사항을 명확히 구분

[전사본]
(녹음 전사 텍스트를 붙여넣으세요)

[출력 형식]
1. 3줄 요약
2. 발언자별 핵심 의견
3. 결정 사항
4. 미결 사항 (추가 논의 필요)
5. 액션 아이템 (담당자, 기한)
```

> 💡 **팁**: 회의 녹음은 Clova Note, Otter.ai 등으로 자동 전사할 수 있습니다. 전사 → AI 정리 → 검토 순서로 진행하면, 회의 직후 5분 안에 회의록을 배포할 수 있습니다.

## 변형: 액션 아이템 추적

```
아래는 지난 회의록입니다.
각 액션 아이템의 진행 상황을 추적할 수 있는 체크리스트를 만들어주세요.

[지난 회의록]
(회의록을 붙여넣으세요)

[출력 형식]
- [ ] [액션 아이템] — 담당: [이름] / 기한: [날짜] / 상태: 미시작
```
