# 보고서 작성 프롬프트 템플릿

> 복사 → [괄호] 수정 → AI에 붙여넣기

---

## 기본형: 업무 보고서

```
당신은 [팀명]의 보고서 작성 전문가입니다.

아래 내용을 바탕으로 [보고서 유형]을 작성해주세요.

[조건]
- 보고 대상: [임원/팀장/팀원]
- 분량: [A4 1장/2장/슬라이드 5장 분량]
- 포함 항목: [항목1, 항목2, 항목3]
- 톤: [간결하고 핵심 위주 / 상세하고 근거 중심]

[내용/데이터]
(여기에 보고할 내용, 수치, 데이터를 붙여넣으세요)
```

---

## 활용 예시

### Before (기존 방식)

**상황**: 마케팅팀 주간 보고서를 작성해야 함. 데이터를 정리하고 문장을 다듬는 데 1시간 소요.

### After (AI 적용)

**프롬프트:**
```
당신은 마케팅팀의 보고서 작성 전문가입니다.

아래 데이터를 바탕으로 주간 마케팅 성과 보고서를 작성해주세요.

[조건]
- 보고 대상: 마케팅 본부장
- 분량: A4 1장 이내
- 포함 항목: 주요 지표 요약, 캠페인별 성과, 전주 대비 변화, 다음 주 액션 아이템
- 톤: 간결하고 핵심 위주. 수치는 표로 정리.

[데이터]
- 총 방문자: 45,230 (전주 대비 +12%)
- 캠페인 A (SNS 광고): CTR 3.2%, 전환 150건, ROAS 320%
- 캠페인 B (검색 광고): CTR 1.8%, 전환 80건, ROAS 210%
- 캠페인 C (이메일): 오픈률 22%, 클릭률 4.5%, 전환 45건
- 신규 리드: 275건 (전주 대비 +8%)
- 이슈: 캠페인 B 성과 하락, 키워드 경쟁 심화 추정
```

**결과**: 10분 만에 초안 완성. 검토/수정 5분. **총 15분** (기존 1시간 → 75% 절감)

## 변형: 주간/월간 정기 보고

```
아래는 이번 [주/월]의 [팀명] 업무 데이터입니다.
지난 [주/월] 보고서 형식과 동일하게 작성해주세요.

[지난 보고서 형식]
(이전 보고서를 그대로 붙여넣으세요)

[이번 데이터]
(새로운 데이터를 붙여넣으세요)

변경사항:
- 전 기간 대비 주요 변화를 하이라이트해주세요
- 특이사항이 있으면 별도 섹션으로 분리해주세요
```

> 💡 **팁**: 이전 보고서를 함께 넣으면 AI가 형식과 톤을 학습합니다. 매번 설명할 필요가 없어집니다.

## 변형: 임원 요약 보고

```
아래 상세 보고서를 임원용 1페이지 요약으로 압축해주세요.

[원칙]
- 핵심 수치 3개만 강조
- "그래서 어떻게 해야 하는가"를 마지막에 명확히
- 전문용어를 비전문가도 이해할 수 있게 풀어서

[상세 보고서]
(여기에 전체 보고서를 붙여넣으세요)
```
