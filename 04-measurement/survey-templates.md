# 팀원 설문 템플릿

> 복사해서 바로 사용할 수 있는 설문 3종입니다. 구글 폼, 타입폼, 노션 등 어떤 도구에든 옮겨서 쓰세요.

---

## 사용 가이드

| 설문 | 시기 | 소요 시간 | 목적 |
|------|------|-----------|------|
| **도입 전 설문** | AX 시작 전 (Baseline) | 3분 | 현재 상태 파악, Before 데이터 |
| **주간 체크인** | 매주 금요일 | 1분 | 사용 현황 추적, 문제 조기 발견 |
| **최종 평가 설문** | 4주차 (또는 시범 종료 시) | 5분 | After 데이터, 성과 증명 |

> **팁**: 설문은 짧을수록 응답률이 높습니다. 선택형 위주로 구성하고, 주관식은 최소화하세요.

---

## 설문 1: 도입 전 설문 (Baseline)

AX 시작 전, 현재 상태를 측정합니다. 이 데이터가 있어야 "얼마나 좋아졌는지"를 증명할 수 있습니다.

```

응답자: [이름 또는 익명 코드]
소속: [부서/팀]
직군: [사무직 / 개발직 / 영업·CS / 관리직 / 기타]
일자: [YYYY-MM-DD]

─────────────────────────────────────

1. 현재 AI 도구를 업무에 사용하고 있습니까?
   □ 전혀 사용하지 않는다
   □ 가끔 사용한다 (월 1~2회)
   □ 정기적으로 사용한다 (주 1회 이상)
   □ 매일 사용한다

2. 사용 중인 AI 도구가 있다면 모두 선택해주세요. (복수 선택)
   □ ChatGPT
   □ Claude
   □ Gemini
   □ GitHub Copilot
   □ Microsoft Copilot
   □ 기타: ___________
   □ 사용하지 않음

3. 아래 업무에 각각 주당 몇 시간을 쓰고 있습니까?
   (해당 업무가 없으면 0으로 표기)

   보고서/문서 작성:      ___ 시간/주
   이메일 작성/답변:      ___ 시간/주
   회의록/메모 정리:      ___ 시간/주
   데이터 분석/정리:      ___ 시간/주
   코드 작성 (해당 시):   ___ 시간/주
   기타 반복 업무:        ___ 시간/주 (업무명:        )

4. AI가 업무에 도움이 될 것이라 생각합니까?
   □ 1 (전혀 그렇지 않다)
   □ 2
   □ 3 (보통)
   □ 4
   □ 5 (매우 그렇다)

5. AI 도입에 대해 우려되는 점이 있다면? (복수 선택)
   □ 보안/정보 유출
   □ 결과물 품질 (부정확할까봐)
   □ 사용법을 모름
   □ 내 일자리에 영향
   □ 추가 업무가 될까봐
   □ 특별한 우려 없음
   □ 기타: ___________
```

## 설문 2: 주간 체크인

매주 금요일, 1분 이내로 응답할 수 있는 짧은 설문입니다. **부담 없는 수준**을 유지하세요.

```
[팀명] AI 활용 주간 체크인 — [N]주차

응답자: [이름 또는 익명 코드]
일자: [YYYY-MM-DD]

─────────────────────────────────────

1. 이번 주에 AI를 업무에 사용했습니까?
   □ 사용하지 않았다
   □ 1~2회 사용했다
   □ 3~4회 사용했다
   □ 거의 매일 사용했다

2. AI를 어떤 업무에 사용했습니까? (복수 선택)
   □ 보고서/문서 작성
   □ 이메일 작성
   □ 회의록 정리
   □ 데이터 분석
   □ 코드 작성/리뷰
   □ 아이디어 브레인스토밍
   □ 번역
   □ 기타: ___________
   □ 사용하지 않음

3. 이번 주 AI 사용 만족도는?
   □ 1 (전혀 도움 안 됨)
   □ 2
   □ 3 (보통)
   □ 4
   □ 5 (매우 도움 됨)

4. (선택) 이번 주 AI 사용 중 좋았던 점 또는 어려웠던 점 한 줄:
   _______________________________________________
```

## 설문 결과 분석 가이드

### 핵심 비교 항목

도입 전 설문과 최종 평가 설문을 비교하여 아래를 산출합니다.

| 비교 항목 | 도입 전 (설문 1) | 도입 후 (설문 3) | 변화 |
|-----------|-----------------|-----------------|------|
| AI 사용 빈도 | 문항 1 | 문항 1 | 증가율 |
| 업무별 소요 시간 | 문항 3 | 문항 3 | 절감률 |
| AI 기대감/만족도 | 문항 4 | 문항 4 | 변화량 |
| 우려 사항 | 문항 5 | 문항 8 | 해소 여부 |

### 주간 트렌드 분석

주간 체크인 데이터를 누적하여 아래를 확인합니다.

- **사용 빈도 추이**: 주차별로 증가하고 있는가, 정체되었는가?
- **사용 업무 다양성**: 적용 업무가 늘어나고 있는가?
- **만족도 추이**: 3점 이하 응답이 줄어들고 있는가?
- **미사용자 변화**: 미사용자가 줄어들고 있는가?

### NPS (추천 지수) 산출

최종 설문 문항 9의 응답으로 NPS를 산출합니다.

```
추천자 (4~5점) 비율 - 비추천자 (1~2점) 비율 = NPS

예시: 추천자 60% - 비추천자 10% = NPS 50
→ NPS 50 이상이면 매우 긍정적
```

## 운영 팁

- **익명성 보장**: 솔직한 응답을 위해 익명으로 실시 (이름 대신 코드 사용 가능)
- **응답 독려**: 팀장이 먼저 응답하고 "저도 했습니다" 공유
- **결과 공유**: 설문 결과를 팀에 투명하게 공유 (부정적 피드백 포함)
- **개선 반영**: 수집된 피드백을 실제로 반영하고, 반영 사실을 공유
- **강제 금지**: 응답을 강제하지 마세요. 미응답 자체도 데이터입니다
